- Frontend: React + Vite
- Backend: Node.js + Express
- Database: DynamoDB
- Auth: AWS Cognito
- Storage: S3
- CDN: CloudFront
- Notifications: SNS + SQS
- Serverless: Lambda
- Monitoring: CloudWatch
